This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.tsx`. The page auto-updates as you edit the file.

## Deploy on Valist

To deploy this project onto Valist, simply run the following:

```bash
npm run build
npm run export
```

This will trigger a `next export`, creating an `out` folder that you can upload to Valist using the web dashboard, CLI, or GitHub Action.

For more information, visit the docs at https://docs.valist.io
